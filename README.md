# Portal de Proyectos - Cámara de Ceuta

Portal interno/público de acceso a proyectos web de la Cámara de Comercio de Ceuta.

## Archivo principal

- `worker.js`: código completo del Cloudflare Worker.

## Despliegue manual en Cloudflare

1. Entrar en Cloudflare Workers.
2. Abrir el Worker del portal.
3. Pulsar `Edit code`.
4. Seleccionar todo (`Ctrl+A`).
5. Pegar el contenido de `worker.js`.
6. Pulsar `Save and deploy`.

## URL actual

- https://portal.camaraceuta.workers.dev/
